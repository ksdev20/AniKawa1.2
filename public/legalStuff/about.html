About This Project

This website is a personal fan-made project inspired by anime streaming platforms like Crunchyroll, but built from scratch using open APIs and public data sources. It is non-commercial and exists to explore UI/UX, design, and development in a real-world project context.

Inspiration

Visually inspired by Crunchyroll, but no code or assets were copied. Layout and logic were built independently for educational purposes.

Data & Video Sources

  • Anime metadata is sourced from AniList and TMDB
  • Videos are embedded only from AniOneAsia’s official YouTube channel
  • We do not host, modify, or download any video files
